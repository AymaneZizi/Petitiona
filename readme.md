## ğŸš€ Usage

1. **Create a Petition:** Go to **Petitions > Add New**
2. **Configure Fields & Settings:** Adjust signature limits, form fields, and email restrictions
3. **Embed on a Page:** Use the shortcode `[petitiona id=X]`
4. **Monitor Signatures:** View statistics in the admin dashboard
5. **Export Data:** Download signatures as CSV when needed
