Create and manage online petitions with customizable forms, signature tracking, and export capabilities.

== Description ==

Petitiona lets you create and manage online petitions directly on your WordPress site. Set signature goals, customize form fields, restrict email domains, and export signature data.

Key Features:

* Customizable petition forms
* Signature goal tracking
* Start/end date scheduling
* Email domain restrictions
* CSV signature export
* Shortcode integration
* Progress bar display
* Mobile-friendly design

== Installation ==

1. Upload 'petitiona' to the '/wp-content/plugins/' directory
2. Activate through the 'Plugins' menu
3. Configure via 'Petitions' in admin menu

== Usage ==

1. Create petition: Petitions > Add New
2. Configure fields and settings
3. Add to page using shortcode: [petitiona id=X]
4. Monitor signatures in admin dashboard
5. Export data as needed

== Frequently Asked Questions ==

= Can I customize form fields? =
Yes, enable/disable and make fields required as needed.

= How do I export signatures? =
Use the Export button in the Signatures view.

= Can I limit who can sign? =
Yes, restrict by email domain (e.g., @university.edu).
